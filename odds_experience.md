# 竞彩欧赔即时数据经验规律

> 来源：连杰实战总结
> 更新时间：2026-05-08

## 盘口变化规律

### 亚盘升降盘规律

| 盘口变化 | 赛果倾向 | 适用联赛 |
|---------|---------|---------|
| 半球 → 半球平 | 平局 | 法甲 |
| 受半 → 受半球平 | 负 | - |
| 平半 → 半 | 不胜（客胜） | 意甲 |
| 平半 → 半 | 负 | 荷兰 |
| 平半 → 半 | 胜 | 葡萄牙 |
| 半一 → 半 | 负 | 西甲 |
| 半 → 半一 | 胜 | 通用 |
| 平 → 平半 | 负 | 通用 |

### 盘口+水位组合规律

| 盘口+水位 | 升降模式 | 赛果 | 备注 |
|-----------|---------|------|------|
| 客2.2几 | - | 胜 | 意甲 |
| 主2.2几 | - | 负 | 意甲 |
| 主2.3几 | - | 胜 | 意甲 |
| 2.25/2.4 | - | 胜 | - |
| 2.35/2.45 | - | 平 | - |
| 2.2 | - | 不胜 | 日乙 |
| 2.1 | - | 不胜 | 日乙 |

### 升降平模式规律

| 升降模式 | 赛果 | 条件 | 联赛 |
|---------|------|------|------|
| 升降升 | 负 | 客低赔 | 通用（i升降升） |
| 升降升 | 负 | - | - |
| 升降降 | 负 | - | 中国 |
| 升降降 | 平 | - | 中国 |
| 升降降 | 胜 | - | 韩 |
| 升降降 | 负 | 客2.1 | 英超 |
| 升降降 | 胜 | - | 英超 |
| 升降降 | 胜 | 让0.75→0.5, 1.1几平 | 英超 |
| 升降降 | 负 | 让1球0.75→0.5, 1.7几主 | 英超 |
| 不变降升 | 负 | - | - |
| 升降升 | 负 | 让0.75, 5.几 | 英超（让球） |

### 赔率区间规律

| 赔率范围 | 升降模式 | 赛果 | 备注 |
|---------|---------|------|------|
| 1.90 | 升降升 | 负 | - |
| 1.8几 | 升降升 | 几乎都是赢 | - |
| 1.1几 | 升降降 | 平 | 一球0.75→0.5 |
| 1.2几 | 升降降 | - | 一球0.75→0.5 |

## 联赛特定规律

### 德甲
- 平 → 平半：都是平局
- 客2.2几：都是胜

### 意甲
- 平半升半：不胜（客胜概率大）
- 客2.2几：胜
- 主2.2几：负
- 主2.3几：胜

### 英超
- 升降降：胜
- 客2.1升降降：客负
- 让0.75降升升：5.几都是负
- 让1球0.75→0.5，1.1几：平
- 让1球0.75→0.5，1.7几主：负

### 西甲
- 半一降半：负

### 日乙
- 2.2：不胜
- 2.1：不胜

### 韩
- 2.1升降降：胜

### 中国
- 升降升：负
- 升降降：平

### 荷兰
- 平半升半：负

### 葡萄牙
- 平半升半：胜

### 法甲
- 半球降半球：平
- 受半降受半球：平

## 使用方式

在 `final_report_generator.py` 生成报告时，自动匹配当前比赛的赔率数据与上述规律，输出匹配度和参考建议。

匹配逻辑：
1. 获取当前比赛的亚盘变化（初盘→即时盘）
2. 获取赔率升降平模式
3. 获取赔率区间
4. 按联赛筛选匹配的规则
5. 输出匹配的规则列表和倾向赛果
